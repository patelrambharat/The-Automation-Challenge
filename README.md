# 🤖 RPA Customer Onboarding Automation (UiPath)

## 📌 Overview
This project demonstrates an end-to-end Robotic Process Automation (RPA) solution built using UiPath to automate customer onboarding. The bot reads customer data from an Excel file and inputs it into both web-based and desktop CRM systems, reducing manual effort and improving accuracy.

---

## 🚀 Features
- 📊 Read and process customer records from Excel using DataTables  
- 🔁 Row-by-row iteration of data  
- 🌐 Automate Web and Desktop applications  
- 🧩 Modular workflow design using multiple XAML files  
- 🎯 Dynamic selector handling for unstable UI elements  
- 🔍 Support for Strict, Fuzzy, and Computer Vision selectors  
- ⚠️ Robust error handling using Try Catch and Retry Scope  
- 🔀 Conditional logic using If and Switch activities  
- ✅ Data validation and verification  

---

## 🧠 Key Learnings
- Built scalable and reusable automation workflows  
- Worked with dynamic selectors and anchors  
- Handled complex UI using Computer Vision  
- Applied best practices for text input automation  
- Understood CAPTCHA limitations and used human-in-the-loop approach
